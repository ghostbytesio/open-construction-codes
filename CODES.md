# Open Construction Codes (OCC) — code catalog

_Version 0.2.0. Generated from `divisions/*.json` — do not hand-edit; regenerate with `node scripts/build-catalog.mjs`._

A code is `DD.NN` — a two-digit **division** and a section number. Section numbers are stable identifiers, not a ranking: they are assigned in order of recognition and never renumbered, so the number does not imply position. Browse below or use your browser's find (Ctrl/⌘-F) to search by code or trade.
